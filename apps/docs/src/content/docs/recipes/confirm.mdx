The Svelte version is the same idea with `<DialogRegister>`:

<Tabs>
  <TabItem label="Svelte">

```svelte
<script>
  import { DialogRegister } from "@dialog-fn/svelte";
  import ConfirmDialog from "./ConfirmDialog.svelte";

  let showDialog;

  async function remove() {
    const confirmed = await showDialog({ message: "This deletes the item." });
    console.log(confirmed ? "confirmed" : "dismissed");
  }
</script>

<button on:click={remove}>Delete item</button>
<DialogRegister dialogComponent={ConfirmDialog} bind:showDialog />
```
